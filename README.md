# Networking-project-for-Netcore

## 📌 Project Summary

This networking assignment focused on designing and securing a small enterprise network using Cisco networking technologies and Access Control Lists (ACLs). The project simulated communication between different departments — Administration, Sales, and Human Resources (HR) — while implementing security rules to control network access and traffic flow.

The assignment involved configuring network topology, testing communication between devices, and applying ACL commands to permit or deny access between departmental PCs. Successful and failed connectivity tests demonstrated how ACLs enhance network security through traffic filtering and segmentation.

---

## 🛠️ Tools & Technologies Used

* Cisco Packet Tracer
* Cisco IOS Command Line Interface (CLI)
* Access Control Lists (ACLs)
* IPv4 Addressing
* Network Topology Design
* Ping & Connectivity Testing
